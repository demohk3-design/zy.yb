# 甲醇 (MA2609) 机构研报汇总与极简推演

## 1. 核心机构研报追踪 (第一梯队 11 家全覆盖)

*   **[十一大机构整体分歧]：** 
    目前第一梯队机构对甲醇主力合约MA2609整体持**宽幅震荡、重心承压**态度（5家看空/偏空，5家看低位震荡，1家看偏多筑底）。核心博弈在于地缘政治（伊朗进口）溢价消退及7月进口到港增量预期（偏空），与动力煤成本支撑及MTO开工率好转预期（偏多/托底）。
*   **[明确给出点位与核心逻辑的机构]：** 
    *   **国泰君安期货：** 认为霍尔木兹海峡地缘溢价退潮，海外供给修复及7月进口量放量（预计回升至70-85万吨）预期主导市场，价格偏空承压，但跌幅将受动力煤支撑而收窄。
    *   **华泰期货：** 港口去库相对抗跌，但内地现货价格走弱。下游MTO（甲醇制烯烃）利润亏损严重抑制了采购积极性，盘面缺乏大涨动力，维持弱势震荡。
    *   **银河期货：** 预计短期基本面缺乏利多题材。下方重要支撑在 **2400-2450元/吨**，上方阻力在 **2700-2800元/吨**。
    *   **瑞达期货：** 地缘风波缓和后海外到港预期增加，预计MA2609合约在 **【2420, 2680】元/吨** 宽幅区间运行，操作上建议区间偏空为主。
    *   **东证期货：** 国内煤炭及原料价格相对抗跌，但高进口压力逐步兑现，建议反弹至2650元/吨上方偏空波段操作。
    *   **中信期货：** 主力合约MA2609近期呈现“港口硬、内地弱”分化，目前2400元/吨成本防线牢固，在煤价未大跌前深跌空间有限，建议观望。
    *   **永安期货：** 供需边际走向宽松，国内煤制利润回归盈亏平衡边缘。建议采取多甲醇空PP（聚丙烯）的强弱套利策略。
    *   **方正中期期货：** 原料动力煤中枢预计在800-850元/吨运行，甲醇制烯烃需求表现分化，盘面多跟随现货价格低位磨底。
    *   **建信期货：** 建议关注生物柴油等边际政策及MTO开工拐点。短期受制于地缘溢价挤出，区间弱势整理。
    *   **五矿期货：** 随着进口放量预期兑现，沿海库存面临重新累积，价格缺乏大幅上行驱动。
    *   **广发期货：** 整体化工板块承压，甲醇以2400元/吨为技术性底部，建议主力合约以2420-2650元/吨区间内逢高布空为主。
*   **[特色产业机构]：** 
    *   **中粮期货（农产品/油脂油料特色，此品种亦有覆盖）：** 指出甲醇近期估值受煤价挺价和地缘政治交织影响。内地高炉与煤制工艺在2400元/吨附近形成完全成本支撑，但华东港口去库已近尾声，7月进口船只集中到港是近端最大压力，不建议在2550元/吨上方追高。

## 2. 基本面数据共识提炼 (基于上述研报)

*   **成本利润底线：** 
    动力煤价格中枢在 **800 - 850元/吨**，折算国内煤制甲醇完全生产成本在 **2400 - 2450 元/吨**。此区间为盘面提供了较强的产业成本防御底线，企业在2400元/吨附近惜售心态明显。
*   **核心多空矛盾：** 
    *   *近端现实（偏空）*：地缘风险降温导致进口预期恢复，7月集中到港压力较大；下游MTO开工受利润压制，买盘以刚需为主。
    *   *远端预期（多空交织）*：下半年煤炭消费旺季（成本托底） vs 进口船只集中到货与国内新增产能达产导致的累库预期。
*   **关键支撑/压力位：** 
    *   *关键支撑区间*：**2400 - 2450 元/吨**（煤制甲醇盈亏线与技术底部重仓区）。
    *   *关键压力区间*：**2680 - 2750 元/吨**（地缘溢价区间阻力与下游MTO买盘承载上限）。

## 3. 综合推演与交易结论 (Agent 策略生成)

*   **单边操作建议：** 
    建议采取**低位宽幅震荡、逢高试空**的波段交易思路。
    *   *做空参考*：当MA2609反弹至 **2650-2700元/吨** 阻力带且进口到港无阻碍时，分批轻仓建空。
    *   *做多/止盈参考*：下探至 **2400-2430元/吨** 强成本区间时空单止盈，可分批轻仓试多。
*   **风控安全垫：** 
    若中东局势突发恶化影响霍尔木兹海峡通航（进口断供风险），或者MA2609日线收盘站稳 **2750元/吨**，空单须立刻离场。

## 4. 风险点预警

*   **中东地缘局势恶化**：伊朗出口船只受阻导致国内港口进口量断崖式下跌。
*   **煤炭价格坍塌**：动力煤价格跌破800元/吨限制，成本端托底效能丧失。
