# 花生 (PK2610) 机构研报汇总与极简推演

*   **[明确给出点位与核心逻辑的机构]：** 
    *   **国泰君安期货：** 认为新季面积减少预期及天气炒作升温提振盘面，但高库存现实对上方反弹空间形成强压，单边维持宽幅震荡整理。
    *   **华泰期货：** 国内花生市场处于“弱现实与强预期”的对抗中。目前油厂及渠道库存处于近五年同期高位，去库缓慢，建议轻仓波段操作。
    *   **银河期货：** 益海油厂提价收购对盘面交割成本形成支撑，短期走势偏强，主力运行参考 **【8100, 8500】元/吨**。
    *   **瑞达期货：** 新季播种面积减少及天气扰动提供升水，预计PK2610在 **【8100, 8480】元/吨** 区间波动，操作以区间震荡偏强为主。
    *   **东证期货：** 指当前现货交收平淡，食品厂采购积极性低。上方受到8500元/吨的强阻力压制，反弹后可分批轻仓试空。
    *   **中信期货：** 随着主力合约移仓换月，资金博弈加剧。PK2610在8000-8100元/吨区间受到仓单成本与挺价心理的双重支撑，下方空间较小。
    *   **永安期货：** 新季天气炒作是未来核心看点。虽然现货基本面偏弱限制涨幅，但在天气炒作退潮前建议以回调偏多思路对待。
    *   **方正中期期货：** 下游油厂成品库存偏高，油用需求力道有限，建议逢高布局空单，上方压力8460-8480元/吨。

    *   **五矿期货：** 基层余货见底，贸易商及农户惜售挺价意愿强烈，短期成本端支撑显现，建议回调试多。
    *   **广发期货：** 螺纹及农产品板块联动偏震荡。花生主力合约在8100元/吨附近企稳，预计震荡中枢上移，区间震荡参与。

## 2. 基本面数据共识提炼 (基于上述研报)

*   **成本利润底线：** 
    新季花生种植成本（种子、化肥、地租等）折合盘面仓单成本在 **8000 - 8100 元/吨** 之间。农户与持货商在该区间惜售挺价意愿极其强烈，构筑了坚实的产业链成本底线。
*   **核心多空矛盾：** 
    *   *近端现实（偏空）*：终端需求不振，食品厂开工率不足；油厂成品库存堆积至五年同期最高水平，去库存压力沉重。
    *   *远端预期（偏多）*：新季全国花生种植面积预期收缩，且7-8月花针期与荚果生长关键期面临干旱/高温等天气炒作。
*   **关键支撑/压力位：** 
    *   *关键支撑区间*：**8000 - 8100 元/吨**（农户成本线与盘面历史技术强支撑带）。
    *   *关键压力区间*：**8460 - 8500 元/吨**（现货高累库阻力位与前期反弹高点密集压制区）。

## 3. 综合推演与交易结论 (Agent 策略生成)

*   **单边操作建议：** 
    建议采取**震荡偏强、逢低试多**的筑底反弹波段思路。
    *   *做多参考*：当PK2610回调下探至 **8100-8150元/吨** 区间企稳时，分批建立轻仓多单，目标看向 8450元/吨 以上。
    *   *做空参考*：在 **8480-8500元/吨** 阻力区间冲高受阻时空单参与，但需严格风控。
*   **风控安全垫：** 
    若盘面日线收盘跌破 **8000 元/吨** 成本支撑线，或者产区天气转为持续风调雨顺导致丰产预期强化，多单必须果断止损。

## 4. 风险点预警

*   **主产区极端异常天气**：7-8月山东、河南等主产区遭遇严重干旱或洪涝灾害，推升减产预期。
*   **油厂停止提价收购**：若益海等头部油厂在库存积压下减少收购或调低收购价，盘面成本支撑将松动。
